Search arXiv for papers related to: $ARGUMENTS

Use the arxiv Python library via:
  uv run --directory ~/Projects/PhenomML/cc-tools python

Search for the top 8–10 most relevant results. For each paper present:
- Title
- Authors (first three, then "et al." if more)
- Date (month year)
- arXiv ID (as a clickable link: https://arxiv.org/abs/XXXX.XXXXX)
- 2–3 sentence summary of the contribution and key findings

Group results by sub-topic if natural clusters exist. After the list, offer to:
1. Fetch and summarize the full PDF of any listed paper (use cc-markitdown on the PDF URL)
2. Search for related work by any of the authors
3. Narrow or broaden the search
